SourceQuery!

Currently in late Alpha, SteamQuery for Powershell gives you 3 simple cmdlets to query any Steam server for numerous datapoints, including: Player list with Steam names, player connection count and connected time, map name, and server OS. Also provides game-specific data such as ingame time, player skins, and available vehicles. Currently tested with ARK: Survival Evolved. Github link coming soon. Inspired by this PHP SourceQuery

PowerGIL!

Currently Build 205, PowerGIL is a Powershell utility kit. It provides numerous useless tools, ranging from making new Powershell windows and resizing current windows, opening Explorer windows, encrypting and decrypting text messages onto files on your HDD, sending and reciving with UDP, watching your clipboard and evne sending it to another PC with UDP! Flip your text into bytes and back! Check out PowerGIL here.

ARKData!

Now in its 1.3 revision, the ARKData player tracker now can track multiple servers simultaneously. Just give it an IP and it does the rest - find the server, get players, time of day, write data files, and stamp self-refreshing website report dashboards every minute. Filling out the .tribe.csv data file is still a manual process, but for a little bit of work (typing out the ARK name and Tribe name) you get crazy more useful data. All written in Powershell.

Check out ARKData's currently-tracked servers now!
